# Personal Life Dashboard

A comprehensive web application designed to help people manage different aspects of their daily life in today's fast-paced world. This modern life cycle project addresses key areas that are crucial for personal well-being and productivity.

## 🌟 Features

### 📊 Dashboard Overview
- **Health Score**: Real-time calculation based on daily metrics (steps, water intake, sleep)
- **Budget Tracking**: Visual representation of monthly spending vs budget
- **Goals Progress**: Track completed and active personal goals
- **Habit Streaks**: Monitor your longest habit streaks
- **Recent Activities**: Timeline of your latest accomplishments

### 💪 Health & Wellness
- **Daily Metrics Tracking**: Steps, water intake, and sleep hours
- **Mood Tracker**: Record and visualize your daily mood with emoji-based system
- **Health Score Calculation**: Automated scoring based on your daily health metrics

### 💰 Financial Management
- **Expense Tracking**: Add and categorize your daily expenses
- **Budget Categories**: Track spending across different categories (food, transport, entertainment, etc.)
- **Budget Overview**: Visual representation of spending vs limits
- **Recent Expenses**: Quick view of your latest transactions

### 🎯 Goals & Objectives
- **Goal Setting**: Create personal, career, health, financial, and educational goals
- **Progress Tracking**: Visual progress bars with percentage completion
- **Deadline Management**: Set and track goal deadlines
- **Goal Categories**: Organize goals by different life areas

### 📅 Habit Tracker
- **Habit Creation**: Add daily, weekly, or monthly habits
- **Streak Tracking**: Monitor consecutive days of habit completion
- **Daily Checklist**: Simple checkbox interface for habit completion
- **Progress Visualization**: See your habit consistency over time

### ⏰ Daily Schedule
- **Event Management**: Add and organize daily events
- **Time-based Scheduling**: Set specific times for events
- **Today's Focus**: View today's scheduled events in chronological order
- **Event Descriptions**: Add detailed descriptions to your events

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. Download or clone the project files
2. Ensure you have the following files in the same directory:
   - `index.html`
   - `styles.css`
   - `script.js`

### Running the Application
1. **Option 1: Direct File Opening**
   - Simply open `index.html` in your web browser
   - All functionality will work locally

2. **Option 2: Local Server (Recommended)**
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```
   - Open your browser and navigate to `http://localhost:8000`

## 📱 How to Use

### First Time Setup
1. Open the application in your browser
2. The dashboard will load with sample data to help you get started
3. Navigate through different sections using the sidebar menu
4. Start adding your own data to personalize the experience

### Daily Workflow
1. **Morning**: Check your schedule and set daily health metrics
2. **Throughout the day**: 
   - Add expenses as they occur
   - Complete habits and update progress
   - Record your mood
3. **Evening**: Review your dashboard and plan for tomorrow

### Data Persistence
- All your data is stored locally in your browser's localStorage
- Data persists between sessions
- No account creation or internet connection required
- Your data remains private on your device
